# Lab 03: Funciones de Cálculo y Lógica

Este laboratorio se centró en implementar un conjunto variado de funciones de ayuda, agrupadas en diferentes archivos.

---

## 1. Ejercicios_simples.py: Funciones de Lógica y Validación

* **Enunciado:** Implementar un conjunto de funciones de validación y cálculos básicos.
    * `es_bisiesto(a)`: Comprueba si un año es bisiesto.
    * `es_par(a)`: Comprueba si un número es par.
    * `último_dígito(a)`: Devuelve el último dígito de un entero.
    * `es_letra(x)`: Comprueba si un carácter es una letra minúscula.
    * `hora(a)`: Valida si un entero de 6 dígitos (hhmmss) es una hora correcta.
    * `coste(precio, porcentaje)`: Calcula el precio final sumando un porcentaje.
* **Mi Solución:** [Ver implementaciones](./Ejercicios_simples.py)

---

## 2. lab03.py: Funciones Matemáticas y Conversiones

* **Enunciado:** Implementar funciones para cálculos matemáticos y conversiones de unidades y tiempo.
    * **Matemáticas:** `es_multiplo(a, b)` y `distancia(x1, y1, x2, y2)`.
    * **Unidades:** `pulgadas_a_cm`, `pies_a_cm` y `centimetros`.
    * **Tiempo:** Funciones para calcular la diferencia entre horas (`dif_total`, `dif_horas`, `dif_minutos`) y para descomponer segundos (`horas`, `minutos`, `segundos`, `horas_minutos_segundos`).
* **Mi Solución:** [Ver implementaciones](./lab03.py)
